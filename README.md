# egret-framework
用于egret的一套框架 包含众多游戏中可能用到的系统

## 当前版本功能

- [x] 简易ECS框架
- [x] A*寻路(AStar)
- [x] 常用碰撞检测
- [x] 数学库
  - [x] 简易矩阵类
  - [x] 简易2d 向量类
  - [x] 掩码实用类
- [x] BreadthFirst 寻路算法
- [x] Dijkstra 寻路算法
- [x] 事件处理器
