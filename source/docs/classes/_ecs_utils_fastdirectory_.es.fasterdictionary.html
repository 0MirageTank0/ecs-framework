Class FasterDictionary<TKey, TValue>

创建这个字典的原因只有一个： 我需要一个能让我直接以数组的形式对值进行迭代的字典，而不需要生成一个数组或使用迭代器。 对于这个目标是比标准字典快N倍。 Faster dictionary在大部分操作上也比标准字典快，但差别可以忽略不计。 唯一较慢的操作是在添加时调整内存大小，因为与标准数组相比，这个实现需要使用两个单独的数组。
