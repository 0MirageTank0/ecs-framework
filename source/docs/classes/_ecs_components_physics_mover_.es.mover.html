Class Mover

辅助类说明了一种处理移动的方法，它考虑了包括触发器在内的所有冲突。 ITriggerListener接口用于管理对移动过程中违反的任何触发器的回调。 一个物体只能通过移动器移动。要正确报告触发器的move方法。

请注意，多个移动者相互交互将多次调用ITriggerListener。
