Class CoroutineManager

基本CoroutineManager。协同程序可以做以下事情:

  • return null(在下一帧继续执行)
  • return Coroutine.waitForSeconds(3)。等待3秒(延时3秒后再次执行)
  • return startCoroutine(another())(在再次执行之前等待另一个协程)
