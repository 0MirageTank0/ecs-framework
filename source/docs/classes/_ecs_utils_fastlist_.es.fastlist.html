Class FastList<T>

围绕一个数组的非常基本的包装，当它达到容量时自动扩展。 注意，在迭代时应该这样直接访问缓冲区，但使用FastList.length字段。

tutorial

for( var i = 0; i <= list.length; i++ ) var item = list.buffer[i];
