Class PriorityQueue<T>

使用堆实现最小优先级队列 O(1)复杂度 这种查找速度比使用字典快5-10倍 但是，由于IPriorityQueue.contains()是许多寻路算法中调用最多的方法，因此尽可能快地实现它对于我们的应用程序非常重要。
